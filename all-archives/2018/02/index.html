Markdown语法和git相关指令

Markdown的基本语法可以参考这篇文章，在博客中主要会用到三种语法结构，

  • 标题控制 用#，#号数量越多，标题级别越低;
  • 超链接： 形如[name](address);
  • 图像: 形如![name](address) 注意感叹号。

而git相关的指令，用于上传图片，作为图床。主要指令有，

  • git add xxx : 添加一个需要push的文件;
  • git commit -m “xxx”: 给这次push添加commitp;
  • git push： 将commit的文件推送到git仓库中。
